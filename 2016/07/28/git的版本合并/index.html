git的版本合并

1.首先保证当前分支的内容已经提交，可执行如下提交

git checkout xxx

git add --all
git commit -m "xxxxxx"
或者
git commit -a

2.将主分支更新到最新

git checkout master
git pull

3.合并合并版本

git checkout xxx
git rebase master (有冲突在此时解决)
git checkout master
git merge xxx (将xxx版本合并到主分支上)

4.合并到远程

git push
